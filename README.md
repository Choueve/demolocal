## 20170912 Git
### Git 指令
- **git init**
```建立本地端repo```

- **git commit -a -m "...."**
```等於 git add + git commit```
- **git clone [遠端repo的URL]**
```將遠端的repo複製到本地端```
- **git add [file name]**
``` 將指定的檔案(或資料夾)加入版控```
```git add . 將全部檔案加入版控(要記得空格再點~)```
- **git log**
```查看commit紀錄```
- **git status**
```查看本地端檔案異動狀態```
- **git rm [file name]**
```刪除工作目錄快取檔案```
```git rm --cached [file name]只會刪除index中的file```

>需要知道VIM基本使用，至少要知道 W(存檔) Q(離開)
